# Photoresistor with LED indicator for Arduino

## Materials
* 1 Arduino Board
* 1 Power Supply
* 1 LED
* 1 Photoresistor (LDR)
* 1 18 ohm resitor
* 1 10k ohm resitor
* Connecting Wires
